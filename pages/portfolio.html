Table Turns 🏆

Go to app GitHub

(Application may take time to load.)
A web application for tracking table football scores with a ranking system. This app was build out of a love for the game to replace a pen and paper score sheet. It is a passion project in constant development and soon to feature the option to register seperate offices (player groups) with login. It is optimized for mobile use as this proved to be the main use in practice.

I started this project from scratch until I had a working prototype to allow myself to learn the basics and rebuild it using claude code. From an educational perspective, I wanted this project to help me better understand the concept of API's and how to setup cloud hosting. I also wanted to learn how to setup a local test environment with a seperate databases without affecting data in the production envirponment.

Inventory Tracker 🧪

After encountering issues with tracking invetory in a lab environment, I wanted to see if I could build a simple desktop application to solve this problem. The application is intentionally build in Python as this is a common language understood by scientists. My goal for this app is to make setup as easy as possible for people with little programming experience. To me this meant, being able to run the app with as little setup as possible, reading and writing data using local CSV files and making configurable settings.

This project is still in progress and will be made available for download when finished.
