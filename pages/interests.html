Professional Interests

Skills and areas I want to develop and grow in.

Agile Project Management

Moving beyond task-level Agile into the bigger picture: learning how to set up projects from scratch, motivate teams to adopt Agile efficiently, and handle the details and nuances of project leadership. Managing my tasks using Agile principles has been a great help to me on a deeply personal level. It helps me organize my thoughts an feel safe in the knowlegde that I canot foget anything important if I follow the process. I want to be able to share this with others and help them experience the same benefits. Once I started reading into the project management side of Agile, I found that there is still a lot I don't quite understand and I definitely don't have the full overview of what is needed to set up a succesful project. In adition to having read about BPMN, this new insight in what I don't know made me want to learn more.

Automated Testing

Learning to write test scripts for automated testing to improve quality assurance processes. I enjoy writing scripts and think automating repetative testing tasks can save time and reduce error. I would like to learn how to use testing tools for regression tests like Selenium and become more proficient and efficient using Postman for API testing.

Technical Understanding

Becoming the person who can translate business needs into technical requirements. I believe the ability to do this well is a unique skill that can make a big difference in project success and I aspire to be that person. To truly understand what is possible from a technical perspective and how to set this up in a realistic and efficient way, I want to gain more experience working closely with developers and learning about the technical side of projects.

Software Development

Though I started learning how to code out of a combination of wanting to contribute to be technical knowledge for the reasons mentioned above and to make my own life easier by automating repetative tasks I encounter in my work as research assistant, I started to really enjoy coding (much to my own surprise). I would like to continue growing in my coding skill and am in particular interested in front-end developement and becoming proficient in JavaScript and utilizing CSS and Bootstrap. In the project where I first started to edit code (DOCCR), I discovered that I really enjoy the puzzle of making a front-end match a design exactly. What initially started as just changing text and fixing spelling, ended in me learning how to create global styles and fixing layouts simply because I started noticing small details and wanting to fix them.
