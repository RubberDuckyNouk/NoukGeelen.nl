# 01 - Project Setup

## Description
Set up the base project structure and integrate Bootstrap.

## Tasks
- Create folder structure (`css/`, `images/`, `pages/`)
- Add Bootstrap via CDN (CSS + JS)
- Create a base HTML template with Bootstrap boilerplate that all pages will follow
- Add a custom `css/style.css` for site-specific styling

## Done when
- A clean folder structure exists
- Bootstrap is linked and working
- A reusable base template is in place
