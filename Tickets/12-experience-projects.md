# 12 - Add Projects to Experience Page

## Description
Add project details from the old WordPress site to the experience page, giving more context about specific work done at each role/client.

## Tasks
- Add project descriptions under the relevant experience entries (Aardevo and/or YellowTwig clients)
- Structure projects clearly using Bootstrap components (nested cards, accordion, or list items)
- Include relevant details per project (client name, role, what was done, technologies used)

## Done when
- Projects from the old site are represented on the experience page
- Each project has enough context to understand the work done
- The page remains clean and easy to scan
